# ABC Pharmacy Medicine Tracker

ABC Pharmacy Medicine Tracker is a Single Page Application for managing medicines and sales records using Angular and ASP.NET Core Web API.

## Description

This application helps track medicines, manage inventory, and record sales. It uses Angular for the frontend and ASP.NET Core Web API for the backend, with JSON files used for server-side storage.

## Project Structure

### Text

```text
PharmacyApp/
├── PharmacyApi/
│   ├── Controllers/
│   ├── Models/
│   ├── Services/
│   ├── Data/
│   └── wwwroot/
└── client-app/
```

## Run the Backend

### Bash

```bash
cd PharmacyApi
dotnet run
```
